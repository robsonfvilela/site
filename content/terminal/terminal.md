### Trem

Mostra um trem passando da direita para a esquerda.
```css
brew install sl
sl
```

### Star Wars: Episode IV — A New Hope

```shell
nc towel.blinkenlights.nl 23
```
### Banner em ASCII
```bash
banner -w 30 TEXTO
```
Para que o texto fique na horizotal, instale o `figlet`.
```console
figlet -w 30 TEXTO
```
Alguns sites oferecem artes de ASCII:
```sh
curl artscene.textfiles.com/asciiart/dragon.txt
```
### Previsão do tempo para os próximos 3 dias
```sh
curl http://wttr.in/
```
### O dia na história
```sh
# Learn famous birthdays on a specific date:
cat /usr/share/calendar/calendar.birthday | grep 03/14
# Learn historical events on a specific date:
cat /usr/share/calendar/calendar.history | grep 06/15
# Learn a musical event on a specific date
cat /usr/share/calendar/calendar.music | grep 10/09
```
